Research Overview

This peer-reviewed research demonstrates advanced biomedical signal processing expertise and contributes to the field of brain-computer interfaces (BCI). The project successfully developed and validated machine learning models capable of distinguishing between motor imagery and actual hand movements using electroencephalography (EEG) signals, with implications for rehabilitation and assistive technology applications.

🧠 Research Goal

Develop robust classification algorithms for neural motor intent recognition. This capability is foundational for neurorehabilitation devices, prosthetic control systems, and brain-computer interface applications used in medical device development.

💡 Clinical Application

The algorithms developed in this research have direct applications in medical devices for stroke rehabilitation, spinal cord injury recovery, and assistive technology for individuals with motor disabilities.

Technical Approach

Signal Processing Pipeline

The project implemented a sophisticated multi-stage signal processing pipeline to extract meaningful features from raw EEG data:

  • Preprocessing: Bandpass filtering (0.5-50 Hz), ICA-based artifact removal, adaptive normalization
  • Feature Engineering: Wavelet packet decomposition, power spectral density analysis, temporal derivatives
  • Dimensionality Reduction: PCA, feature selection techniques to optimize model efficiency

LSTM Architecture

A Long Short-Term Memory (LSTM) neural network was developed to capture temporal dependencies in EEG signals:

  • Model Design: Bidirectional LSTM with attention mechanisms
  • Training Strategy: Cross-validation, regularization, hyperparameter optimization
  • Evaluation: Accuracy, sensitivity, specificity, ROC-AUC analysis
