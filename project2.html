Mind to Motion

EEG-Based Classification of Motor Imagery and Actual Hand Movements Using LSTM Models

Published in
2023 Biomedical Engineering International Conference (BMEiCON-2023)
Authors
Apoorva Sunil Chakkamallisery, Yutthana Pititheeraphab, Sonam Tenzin Pelmo, Tasawan Puttasakul, Thanate Angsuwatanakul, Thapanee Khemanuwong
Affiliation
Rangsit University & King Mongkut's Institute of Technology Ladkrabang

Research Objective

Problem: 15% of global population experiences disabilities. Arm amputees constitute ~30% of all amputees worldwide. Current prosthetic limbs lack naturalness and require complex, invasive procedures.

Solution: Develop EEG-based classification model to distinguish motor imagery from actual hand movements using LSTM deep learning.

Clinical Application: Enable brain-computer interfaces (BCIs) for intuitive prosthetic arm control and neurorehabilitation technologies.

Phase 1: EEG Analysis

Hypothesis: No significant differences in brain wave activity between motor imagery and real motor actions.

Methodology:

  • 21-channel electrode cap acquisition
  • Time-frequency analysis in EEGLAB
  • Feature extraction: Mean power & frequency
  • Analysis across alpha, beta, theta, delta bands

Result: EEG patterns confirmed hypothesis - motor imagery and real actions show similar brain activity.

Brainwave Frequencies

Brainwave Type Frequency Band State
Beta Wave 13-30 Hz Alertness, focused action
Alpha Wave 8-12 Hz Resting state, contemplation
Theta Wave 4-7 Hz Sleep, relaxation
Delta Wave 0.5-3 Hz Deep meditation, sleep

Phase 2: LSTM Classification

Methodology: Recurrent Neural Network (LSTM) for sequential EEG data processing.

Dataset:

  • 8 EEG channels: Fp1, Fp2, F3, F4, C3, C4, P3, P4
  • 4 tasks: Open-eye visualization, Closed-eye visualization, Physical action, Resting
  • 100 total instances
  • Train:Test:Validation = 8:1:1 split (80:10:10)

LSTM Network Architecture

  • Sequence Input Layer: [8×8] matrix processing
  • 2 LSTM Layers: Hidden units = 28 each
  • Fully Connected Layer: Feature consolidation
  • SoftMax Layer: Probability distribution
  • Classification Layer: Final prediction

Hardware & Software Setup

Hardware: OpenBCI Cyton Board (8-channel, 250 Hz sampling), 21-channel Ag/AgCl electrode cap, OpenBCI USB Dongle

Software: OpenBCI GUI v5.0.6, MATLAB R2019a, EEGLAB v2022.0, Lab Streaming Layer, App Designer

Key Implementation: MATLAB AppDesigner GUI developed for real-time data visualization and classification testing using Lab Streaming Layer (LSL) for time-synchronized data collection

Future Scope & Recommendations

Immediate Applications: Integration with prosthetic arm control systems using real-time LSTM classification for intuitive brain-controlled prosthetics.

Critical Next Steps:

  • Personalization: Individual user model training (EEG varies significantly between individuals)
  • Periodic recalibration to account for temporal signal drift
  • Expansion to multi-user validation studies
  • Real-world prosthetic integration testing

Long-term Vision: Seamless EEG-based mind control for individuals with motor impairments, significantly enhancing quality of life and autonomy through advanced BCI technology.
